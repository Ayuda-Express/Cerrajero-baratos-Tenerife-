Instrucciones rápidas para publicar gratis en GitHub Pages (desde el móvil):
1. Crea un repositorio en github.com ( botón New )
2. Sube los dos archivos con "Add file -> Upload files"
3. Commit changes
4. Settings -> Pages -> Source: main branch / root -> Save
5. Espera unos minutos y tu web estará en: https://<tuusuario>.github.io/<repositorio>

Instrucciones rápidas para publicar en Netlify (drag & drop):
1. Entra en netlify.com y crea cuenta (Continue with Google)
2. En el panel, "Add new site" -> "Deploy manually" -> arrastra index.html y styles.css (puedes meterlos en un zip)
3. Publica y cambia el nombre del sitio si lo deseas

Si quieres, te guío en tiempo real para publicar en cualquiera de las dos plataformas.
